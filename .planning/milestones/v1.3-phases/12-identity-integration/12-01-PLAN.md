1. `COLOR_PALETTE` — the same 15-color array used in all four cursor components (copy from CursorCanvas.tsx). Keep private (no export).

2. `identityToColor(identifier: string): string` — takes email (or clientId as fallback). Uses the same djb2 hash algorithm currently in all components:
```
let hash = 0;
for (let i = 0; i < identifier.length; i++) {
  hash = identifier.charCodeAt(i) + ((hash << 5) - hash);
}
return COLOR_PALETTE[Math.abs(hash) % COLOR_PALETTE.length];
```

3. `identityToInitials(displayName: string): string` — implements the decisions from 12-CONTEXT.md:
- If displayName contains a space: first char of first word + first char of last word (split on whitespace, take [0][0] and [last][0])
- If no space: first two chars
- Return uppercase, max 2 chars
- Example: "Jane Doe" → "JD", "jane" → "JA", "j" → "J"
- Handle empty string: return "?"

Add a JSDoc comment block at the top explaining this is the Phase 12 identity utility — color uses email for stable cross-session consistency; initials parse displayName.
