<objective>
Create `LoginForm` and `SignupForm` React components that render Cognito auth UI.
These are pure presentational components: they receive `useAuth` callbacks as props and
display auth state (loading, error) — no internal hook calls.

Purpose: App.tsx (plan 11-03) will render these components when `auth.status === 'unauthenticated'`,
toggling between them with a local boolean. The components need no knowledge of how auth works.

Output: Two new components in `frontend/src/components/`.

  error: string | null
