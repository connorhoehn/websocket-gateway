Create Redis-backed token bucket rate limiter for distributed rate limiting across multiple gateway instances:

1. Implement `RateLimiter` class that:
   - Accepts redis client and logger in constructor
   - Tracks per-client message rate with differentiated limits:
     * General messages: 100 msgs/sec
     * Cursor messages: 40 msgs/sec
   - Uses atomic Redis INCR + EXPIRE for distributed tracking

2. Core method: `async checkLimit(clientId, messageType)`:
   ```javascript
   const limit = messageType === 'cursor' ? 40 : 100;
   const key = `rate:${clientId}:${messageType}`;
   const current = await this.redis.incr(key);
   if (current === 1) {
     await this.redis.expire(key, 1); // 1 second window
   }
   if (current > limit) {
     this.logger.warn(`Rate limit exceeded: ${clientId} ${messageType} ${current}/${limit}`);
     return { allowed: false, current, limit };
   }
   return { allowed: true, current, limit };
   ```

3. Detect message type from payload:
   - `service === 'cursor'` → cursor message (40/sec limit)
   - Everything else → general message (100/sec limit)

4. No penalty/cooldown logic yet - just track and reject. Backpressure response handled in message router.

2. In WebSocket upgrade handler (BEFORE auth check):
   ```javascript
   const clientIp = request.socket.remoteAddress;

   // Check global limit
   if (this.totalConnections >= this.MAX_TOTAL_CONNECTIONS) {
     socket.write('HTTP/1.1 503 Service Unavailable\r\n\r\n');
     socket.destroy();
     return;
   }

   // Check per-IP limit
   const ipCount = this.connectionsByIp.get(clientIp) || 0;
   if (ipCount >= this.MAX_CONNECTIONS_PER_IP) {
     socket.write('HTTP/1.1 429 Too Many Requests\r\n\r\n');
     socket.destroy();
     return;
   }

   // Increment counters
   this.totalConnections++;
   this.connectionsByIp.set(clientIp, ipCount + 1);

   async handleMessage(client, rawMessage) {
     try {
       // Parse
       const message = JSON.parse(rawMessage);

       // Validate structure
       this.validator.validateStructure(message);

       // Validate size
       this.validator.validatePayloadSize(message);

       // Validate channel name if present
       if (message.channelId) {
         this.validator.validateChannelName(message.channelId);
       }

       // Check rate limit
       const messageType = message.service === 'cursor' ? 'cursor' : 'general';
       const rateLimitResult = await this.rateLimiter.checkLimit(client.clientId, messageType);
       if (!rateLimitResult.allowed) {
         this.sendError(client, 'RATE_LIMIT_EXCEEDED',
           `Rate limit exceeded: ${rateLimitResult.current}/${rateLimitResult.limit} msgs/sec`);
         return;
       }

       // Route to service (existing logic)
       await this.routeToService(client, message);

     } catch (error) {
       if (error instanceof ValidationError) {
         this.sendError(client, error.code, error.message);
       } else {
         this.logger.error(`Message handling error: ${error.message}`);
         this.sendError(client, 'INTERNAL_ERROR', 'Failed to process message');
       }
     }
   }
   ```

4. Extend `sendError()` method if needed to support error code + message format
