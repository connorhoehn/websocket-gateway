Gateway CRDT protocol (from STATE.md decision log, Phase 04-01):
```
Subscribe:   { service: 'crdt', action: 'subscribe',   channel }
Unsubscribe: { service: 'crdt', action: 'unsubscribe', channel }
Update:      { service: 'crdt', action: 'update',      channel, update: string (base64 Y.js binary update) }

Server → client events:
crdt:subscribed  { type: 'crdt:subscribed', channel }
crdt:snapshot    { type: 'crdt:snapshot',   channel, snapshot: string (base64 cumulative Y.js state) }
crdt:update      { type: 'crdt:update',     channel, update: string (base64), clientId }
```

Y.js API (install: npm install yjs in frontend/):
```typescript
import * as Y from 'yjs'
// Create shared document and shared text type
const ydoc = new Y.Doc()
const ytext = ydoc.getText('content')  // named shared type

// Encode full document state as binary Uint8Array
import { encodeStateAsUpdate, applyUpdate } from 'yjs'
const updateBytes: Uint8Array = encodeStateAsUpdate(ydoc)
const b64: string = Buffer.from(updateBytes).toString('base64')

// Apply an incoming binary update (e.g. from crdt:snapshot or crdt:update)
const bytes = Buffer.from(b64, 'base64')
applyUpdate(ydoc, bytes)

// Observe text changes (fires after any local or remote update)
ytext.observe(() => {
  const currentText: string = ytext.toString()
})

// Insert/delete text at position (for local edits)
ydoc.transact(() => {
  ytext.delete(0, ytext.length)   // clear
  ytext.insert(0, 'new content')  // insert at position 0
})
```

Ref-based stable callback pattern (from useChat.ts / usePresence.ts):

const sendMessageRef = useRef(sendMessage);

useEffect(() => { currentChannelRef.current = currentChannel; }, [currentChannel]);

// Subscribe effect — gated on connectionState === 'connected'

  if (connectionState !== 'connected' || !currentChannel) return;
  sendMessage({ service: 'crdt', action: 'subscribe', channel: currentChannel });
  return () => {
    sendMessageRef.current({ service: 'crdt', action: 'unsubscribe', channel: currentChannel });
  };
}, [currentChannel, connectionState]);
// sendMessage intentionally excluded — use sendMessageRef for stable access

// onMessage handler effect — separate from subscribe effect
useEffect(() => {
  const unregister = onMessage((msg) => { /* handle */ });
  return unregister;
}, [onMessage]);

    Implementation shape (useCRDT.ts):
    ```typescript
    import * as Y from 'yjs';
    import { encodeStateAsUpdate, applyUpdate } from 'yjs';

    export function useCRDT(options: UseCRDTOptions): UseCRDTReturn {
      const { sendMessage, onMessage, currentChannel, connectionState } = options;
      const [content, setContent] = useState<string>('');

      // Y.Doc lives in a ref — stable across renders, one doc per hook instance
      const ydoc = useRef<Y.Doc>(new Y.Doc());
      const ytext = useRef<Y.Text>(ydoc.current.getText('content'));

      // Stable refs for closures
      const sendMessageRef = useRef(sendMessage);
      // ... currentChannelRef pattern ...

      // onMessage handler effect (separate from subscribe — survives channel changes)
      // On crdt:snapshot: reset ydoc, apply snapshot bytes, setContent
      // On crdt:update: apply update bytes, setContent (Y.js merges automatically)

      // Subscribe effect (gated on connectionState === 'connected')
      // On subscribe: reset ydoc (new Y.Doc()), re-init ytext ref, setContent('')
      // On cleanup: send unsubscribe

      // ytext.observe: setContent(ytext.current.toString())
      // Register observer after doc reset so it reflects correct doc instance

      // applyLocalEdit: encode full state as update, send to gateway, setContent
      const applyLocalEdit = useCallback((newText: string) => {
        ydoc.current.transact(() => {
          ytext.current.delete(0, ytext.current.length);
          ytext.current.insert(0, newText);
        });
        const update = encodeStateAsUpdate(ydoc.current);
        const b64 = Buffer.from(update).toString('base64');
        sendMessageRef.current({
          service: 'crdt',
          action: 'update',
          channel: currentChannelRef.current,
          update: b64,
        });
      }, []);

      return { content, applyLocalEdit };
    }
    ```
