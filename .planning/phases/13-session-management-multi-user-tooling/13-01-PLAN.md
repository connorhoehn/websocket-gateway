<objective>
Add proactive token refresh and multi-tab session sync to useAuth. When the ID token is close to expiry, a setTimeout fires 2 minutes early and calls cognitoUser.refreshSession. If it succeeds, idToken state updates (triggering useWebSocket reconnect automatically). If it fails, the user is signed out with a session-expired message. A BroadcastChannel('auth') propagates TOKEN_REFRESHED and SIGNED_OUT events to all other tabs.

Purpose: Ensures the gateway connection stays live across the full Cognito token lifetime without user re-login. Handles session expiry with clear feedback rather than a silent disconnect.

       ```
       function scheduleTokenRefresh(idToken: string, onRefresh: () => void): ReturnType<typeof setTimeout> | null {
         try {
           const payload = JSON.parse(atob(idToken.split('.')[1].replace(/-/g, '+').replace(/_/g, '/')));
           const expiresInMs = payload.exp * 1000 - Date.now();
           const refreshInMs = expiresInMs - 2 * 60 * 1000; // 2 min early
           if (refreshInMs <= 0) { onRefresh(); return null; }
           return setTimeout(onRefresh, refreshInMs);
         } catch {
           return null;
         }
       }
       ```

    2. REFRESH EXECUTION (useCallback, add to hook body):
       - Read refresh token from localStorage (STORAGE_REFRESH_TOKEN)
       - If missing, call signOut() and set error 'Your session has expired. Please sign in again.'
       - Construct: const refreshToken = new CognitoRefreshToken({ RefreshToken: storedToken })
       - Get current user: const cognitoUser = userPool.getCurrentUser()
       - If no cognitoUser, call signOut with session-expired error
       - Call cognitoUser.refreshSession(refreshToken, (err, session) => {
           if (err || !session) {
             signOut();
             setState(prev => ({ ...prev, status: 'unauthenticated', error: 'Your session has expired. Please sign in again.' }));
             return;
           }
           const newIdToken = session.getIdToken().getJwtToken();
           const newRefreshToken = session.getRefreshToken().getToken();
           localStorage.setItem(STORAGE_ID_TOKEN, newIdToken);
           localStorage.setItem(STORAGE_REFRESH_TOKEN, newRefreshToken);
           setState(prev => ({ ...prev, idToken: newIdToken }));
           broadcastChannel.current?.postMessage({ type: 'TOKEN_REFRESHED', idToken: newIdToken });
           // Re-schedule next refresh
           timerRef.current = scheduleTokenRefresh(newIdToken, doRefresh);
         })
