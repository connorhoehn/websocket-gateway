**POST /invite** — invite a user (GRUP-03)
Route: `groupMembersRouter.post('/invite', ...)`

Logic:
1. Read `userId` (target) from `req.body`; return `400 { error: 'userId is required' }` if missing
2. Prevent self-invite: if `userId === req.user!.sub` return `400 { error: 'Cannot invite yourself' }`
3. `GetCommand` from `social-groups` for `{ groupId: req.params.groupId }`; return `404 { error: 'Group not found' }` if missing
4. Get caller membership via `getCallerMembership(req.params.groupId, req.user!.sub)`
5. If caller is NOT a member OR caller's `role` is neither `'owner'` nor `'admin'`: return `403 { error: 'Only group owners and admins can invite members' }`
   - Note: treat absence of membership record as "not a member"; treat missing `role` as `'member'`
6. Check if target user is already a member: `getCallerMembership(req.params.groupId, userId)` — if the record exists AND `status` is `'active'` (or `status` is undefined): return `409 { error: 'User is already a member' }`
7. Write invite record to `social-group-members`:
   ```
   {
     groupId: req.params.groupId,
     userId,
     role: 'member',
     status: 'invited',
     invitedAt: new Date().toISOString(),
     joinedAt: '',
   }
   ```
8. Return `201 { groupId, userId, status: 'invited' }`

---

**POST /invitations/:action** — accept or decline an invitation (GRUP-04)
Route: `groupMembersRouter.post('/invitations/:action', ...)`

Valid actions: `'accept'` | `'decline'`

Logic:
1. Read `action` from `req.params.action`; return `400 { error: 'action must be accept or decline' }` if not one of those two values
2. `GetCommand` from `social-groups` for `{ groupId: req.params.groupId }`; return `404` if not found
3. Get caller's membership record via `getCallerMembership(req.params.groupId, req.user!.sub)`
4. If no record or `status !== 'invited'`: return `404 { error: 'No pending invitation found' }`
5. If `action === 'accept'`:
   - `UpdateCommand` on `social-group-members`:
     - Key: `{ groupId: req.params.groupId, userId: req.user!.sub }`
     - `UpdateExpression: 'SET #s = :s, joinedAt = :j'`
     - `ExpressionAttributeNames: { '#s': 'status' }`  (status is a reserved word)
     - `ExpressionAttributeValues: { ':s': 'active', ':j': new Date().toISOString() }`
   - Return `200 { message: 'Invitation accepted', groupId: req.params.groupId }`
6. If `action === 'decline'`:
   - `DeleteCommand` from `social-group-members` Key: `{ groupId: req.params.groupId, userId: req.user!.sub }`
   - Return `200 { message: 'Invitation declined' }`

**POST /join** — join a public group without invitation (GRUP-06)
Route: `groupMembersRouter.post('/join', ...)`

Logic:
1. `GetCommand` from `social-groups` for `{ groupId: req.params.groupId }`; return `404` if not found
2. If `group.visibility === 'private'`: return `403 { error: 'This group is private. You must be invited to join.' }`
3. Get caller's existing membership via `getCallerMembership(req.params.groupId, req.user!.sub)`
4. If record exists and `status === 'active'` (or status is undefined): return `409 { error: 'Already a member of this group' }`
5. Write active member record to `social-group-members`:
   ```
   {
     groupId: req.params.groupId,
     userId: req.user!.sub,
     role: 'member',
     status: 'active',
     joinedAt: new Date().toISOString(),
   }
   ```
6. Return `201 { groupId: req.params.groupId, userId: req.user!.sub, role: 'member' }`

---

**DELETE /leave** — leave a group (GRUP-07)
Route: `groupMembersRouter.delete('/leave', ...)`

Logic:
1. `GetCommand` from `social-groups` for `{ groupId: req.params.groupId }`; return `404` if not found
2. Get caller membership; return `404 { error: 'You are not a member of this group' }` if no record or status is 'invited'
3. If caller's `role === 'owner'`: return `403 { error: 'Group owner cannot leave. Transfer ownership or delete the group.' }`
4. `DeleteCommand` from `social-group-members` Key: `{ groupId: req.params.groupId, userId: req.user!.sub }`
5. Return `200 { message: 'Left group successfully' }`

**GET /members** — list all members with roles (GRUP-08, GRUP-09)
Route: `groupMembersRouter.get('/members', ...)`

Logic:
1. `GetCommand` from `social-groups` for `{ groupId: req.params.groupId }`; return `404` if not found
2. Get caller membership; if group is `private` AND caller is NOT an active member: return `403 { error: 'This group is private' }`
3. `QueryCommand` from `social-group-members`:
   - `KeyConditionExpression: 'groupId = :gid'`
   - `ExpressionAttributeValues: { ':gid': req.params.groupId }`
   - `FilterExpression: '#s = :active OR attribute_not_exists(#s)'`
   - `ExpressionAttributeNames: { '#s': 'status' }`
   (This returns only active members, excluding pending invites)
4. Map items to `{ userId, role, joinedAt }` — do NOT expose `status` or `invitedAt` in the response
5. Return `200 { members: [...] }`
