**POST /api/groups** — create a group (GRUP-01)

Validation:
- `name` is required; reject with `400 { error: 'name is required (max 100 chars)' }` if missing or length > 100
- `description` is optional; reject with `400 { error: 'description must be 500 chars or fewer' }` if provided and length > 500
- `visibility` is optional; default `'public'`; reject with `400 { error: 'visibility must be public or private' }` if provided and not one of those two values

Logic:
1. Generate `groupId = uuidv4()`
2. Build `GroupItem` with `ownerId = req.user!.sub`, `createdAt = new Date().toISOString()`, `updatedAt = createdAt`
3. `PutCommand` to `social-groups` with the item
4. `PutCommand` to `social-group-members` with `{ groupId, userId: req.user!.sub, role: 'owner', joinedAt: createdAt }`
5. Return `201` with `{ ...groupItem, role: 'owner' }`

---

**GET /api/groups/:groupId** — get group details

Logic:
1. `GetCommand` from `social-groups` for `{ groupId }`; return `404 { error: 'Group not found' }` if missing
2. `GetCommand` from `social-group-members` for `{ groupId, userId: req.user!.sub }` to check membership
3. If group is `private` and caller is NOT a member: return `403 { error: 'This group is private' }`
4. If group is `public` OR caller is a member: return `200` with the group item (including caller's `role` if member, else `null`)

---

**DELETE /api/groups/:groupId** — delete a group (GRUP-02)

Logic:
1. `GetCommand` from `social-groups` for `{ groupId }`; return `404` if not found
2. Check `item.ownerId === req.user!.sub`; return `403 { error: 'Only the group owner can delete this group' }` if not owner
3. `DeleteCommand` from `social-groups` for `{ groupId }`
4. Return `200 { message: 'Group deleted' }`

Note: Do NOT delete all group members in this plan — that cleanup is out of scope for plan 01.

---

**PATCH /api/groups/:groupId/visibility** — update visibility (GRUP-05)

Logic:
1. Read `visibility` from `req.body`; validate it is `'public'` or `'private'`; return `400 { error: 'visibility must be public or private' }` otherwise
2. `GetCommand` from `social-groups` for `{ groupId }`; return `404` if not found
3. Check `item.ownerId === req.user!.sub`; return `403 { error: 'Only the group owner can change visibility' }` if not owner
4. `UpdateCommand` on `social-groups`:
   - `UpdateExpression: 'SET visibility = :v, updatedAt = :u'`
   - `ExpressionAttributeValues: { ':v': visibility, ':u': new Date().toISOString() }`
   - `ReturnValues: 'ALL_NEW'`
5. Return `200` with `result.Attributes`
