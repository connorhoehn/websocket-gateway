**postLikesRouter** — export, mergeParams:true, mounted at `/rooms/:roomId/posts/:postId`:

POST `/likes` (REAC-01 — like a post):
  - Params: roomId, postId from req.params; userId = req.user!.sub
  - Membership gate: GetCommand on ROOM_MEMBERS_TABLE Key {roomId, userId} → 403 if missing
  - Post existence: GetCommand on POSTS_TABLE Key {roomId, postId} → 404 if missing
  - targetId = `post:${postId}`
  - PutCommand on LIKES_TABLE Item {targetId, userId, type: 'like', createdAt: new Date().toISOString()}
    with ConditionExpression: 'attribute_not_exists(#uid)' and ExpressionAttributeNames: {'#uid': 'userId'}
    → catch ConditionalCheckFailedException → 409 {error: 'Already liked'}
  - 201 {targetId, userId, type: 'like', createdAt}

DELETE `/likes` (REAC-02 — unlike a post):
  - Params: roomId, postId; userId = req.user!.sub
  - Membership gate as above → 403
  - targetId = `post:${postId}`
  - GetCommand on LIKES_TABLE Key {targetId, userId} → 404 {error: 'Like not found'} if missing
  - DeleteCommand on LIKES_TABLE Key {targetId, userId}
  - 204 no body

GET `/likes` (REAC-06 — who liked a post with display names and count):
  - Params: roomId, postId; callerId = req.user!.sub
  - Membership gate → 403
  - Post existence → 404
  - targetId = `post:${postId}`
  - QueryCommand on LIKES_TABLE KeyConditionExpression: 'targetId = :tid' with ExpressionAttributeValues {':tid': targetId}
    FilterExpression: '#t = :like' ExpressionAttributeNames {'#t': 'type'} ExpressionAttributeValues {':like': 'like'}
    (Add '#t' to ExpressionAttributeNames alongside any others already defined)
  - Collect userIds from result.Items
  - If userIds.length > 0: BatchGetCommand on PROFILES_TABLE
    Keys: userIds.map(uid => ({userId: uid}))
    Extract displayName (or fall back to userId) from each profile Item
  - 200 {count: userIds.length, likedBy: [{userId, displayName}]}

**commentLikesRouter** — export, mergeParams:true, mounted at `/rooms/:roomId/posts/:postId/comments/:commentId`:

POST `/likes` (REAC-03 — like a comment):
  - Params: roomId, postId, commentId; userId = req.user!.sub
  - Membership gate on ROOM_MEMBERS_TABLE Key {roomId, userId} → 403
  - Comment existence: GetCommand on COMMENTS_TABLE Key {postId, commentId} → 404
  - targetId = `comment:${commentId}`
  - PutCommand on LIKES_TABLE Item {targetId, userId, type: 'like', createdAt: new Date().toISOString()}
    with ConditionExpression: 'attribute_not_exists(#uid)' ExpressionAttributeNames {'#uid': 'userId'}
    → catch ConditionalCheckFailedException → 409 {error: 'Already liked'}
  - 201 {targetId, userId, type: 'like', createdAt}

DELETE `/likes` (REAC-04 — unlike a comment):
  - Params: roomId, postId, commentId; userId = req.user!.sub
  - Membership gate → 403
  - targetId = `comment:${commentId}`
  - GetCommand on LIKES_TABLE Key {targetId, userId} → 404 if missing
  - DeleteCommand on LIKES_TABLE Key {targetId, userId}
  - 204 no body

**Error logging convention for all handlers:**
  catch (err) {
    // Check for ConditionalCheckFailedException before generic 500
    if ((err as {name?: string}).name === 'ConditionalCheckFailedException') {
      res.status(409).json({ error: 'Already liked' });
      return;
    }
    console.error('[likes] handler error:', err);
    res.status(500).json({ error: 'Internal server error' });
  }

Note: `type` is a DynamoDB reserved word. When using it in FilterExpression, always guard with ExpressionAttributeNames e.g. `'#t': 'type'`.
