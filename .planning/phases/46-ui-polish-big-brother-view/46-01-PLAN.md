Add inline error display to every form that performs a mutation. Use a consistent pattern across all four components:

**Error display pattern (same in every component):**
- Add `const [formError, setFormError] = useState<string | null>(null)` state
- Before each mutation call: `setFormError(null)` to clear previous error
- Wrap mutation in try/catch: `catch (err) { setFormError(err instanceof Error ? err.message : 'Operation failed'); }`
- Render error div after the form controls:
  ```tsx
  {formError && (
    <div style={{ color: '#dc2626', fontSize: 13, marginTop: 6, fontFamily: 'system-ui, -apple-system, sans-serif' }}>
      {formError}
    </div>
  )}
  ```
