   Use `display: 'grid', gridTemplateColumns: '280px 1fr', gap: 16` inside the section card.

   **Left column -- Room Stats:**
   - Header: "Rooms" with count badge
   - List each room from `rooms` prop: room name, type badge (same styling as RoomList.tsx RoomRow), and a "members" indicator. Since we don't have per-room member counts without extra API calls, show the room type and created date.
   - If `rooms.length === 0`, show "No rooms yet" empty state.

   **Right column -- Live Event Feed:**
   - Header: "Live Events" with green dot when `isLive` (same pattern as ActivityPanel)
   - Scrolling list of activity items using `formatActivity` for icon+text and `relativeTime` for timestamps
   - `maxHeight: 400, overflowY: 'auto'` for the feed container
   - If loading, show "Loading events..." centered text
   - If no items, show "No events yet -- run a simulation to see activity here"

4. **Top stats bar** (above the grid):
   - Three inline stat boxes in a flex row:
     - "Online" -- `presenceUsers.length` with a green dot
     - "Rooms" -- `rooms.length`
     - "Events" -- `items.length` (from useActivityFeed)
   - Each stat box: `background: '#f8fafc', border: '1px solid #e2e8f0', borderRadius: 8, padding: '8px 16px', textAlign: 'center'`
