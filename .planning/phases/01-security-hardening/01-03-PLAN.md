Fix presence service memory leak by adding TTL cleanup for stale clients:

1. In constructor, add cleanup interval:
   ```javascript
   this.clientPresence = new Map(); // Already exists
   this.STALE_THRESHOLD = 90000; // 90 seconds
   this.CLEANUP_INTERVAL = 30000; // 30 seconds

   this.cleanupInterval = setInterval(() => {
     this.cleanupStaleClients();
   }, this.CLEANUP_INTERVAL);
   ```

2. Implement cleanupStaleClients method:
   ```javascript
   cleanupStaleClients() {
     const now = Date.now();
     let cleaned = 0;

     for (const [clientId, entry] of this.clientPresence.entries()) {
       if (now - entry.lastHeartbeat > this.STALE_THRESHOLD) {
         this.clientPresence.delete(clientId);
         cleaned++;
         this.logger.debug(`Cleaned up stale presence: ${clientId}`);
       }
     }

     if (cleaned > 0) {
       this.logger.info(`Presence cleanup: removed ${cleaned} stale clients`);
     }
   }
   ```

   - When client sends heartbeat/presence update: `entry.lastHeartbeat = Date.now()`

4. Add shutdown method to clear interval:
   ```javascript
   shutdown() {
     clearInterval(this.cleanupInterval);
     this.logger.info('Presence service shutdown complete');
   }
   ```

5. Call shutdown in server's graceful shutdown handler (already exists via NodeManager)

CRITICAL: Must track lastHeartbeat timestamp for cleanup to work. Update all presence update paths.

Fix chat service memory leak by implementing LRU cache for channel history:

1. Import lru-cache at top:
   ```javascript
   const LRU = require('lru-cache');
   ```

2. In constructor, replace channelHistory Map with per-channel LRU caches:
   ```javascript
   // Old: this.channelHistory = new Map();
   this.channelCaches = new Map(); // channelId -> LRU cache
   this.MAX_MESSAGES_PER_CHANNEL = 100;
   ```

3. Add getChannelCache helper:
   ```javascript
   getChannelCache(channelId) {
     if (!this.channelCaches.has(channelId)) {
       const cache = new LRU({
         max: this.MAX_MESSAGES_PER_CHANNEL,
         updateAgeOnGet: false,
         updateAgeOnHas: false
       });
       this.channelCaches.set(channelId, cache);
     }
     return this.channelCaches.get(channelId);
   }
   ```

4. Update all channelHistory access to use LRU:
   - When adding message: `this.getChannelCache(channelId).set(messageId, message)`
   - When retrieving history: `Array.from(this.getChannelCache(channelId).values())`

5. Optional: Add periodic cleanup to delete empty channel caches:
   ```javascript
   // In constructor
   setInterval(() => {

       if (cache.size === 0) {
         this.channelCaches.delete(channelId);
       }
     }
   }, 300000); // Every 5 minutes
   ```

LRU automatically evicts oldest messages when max is reached - no manual cleanup needed.
