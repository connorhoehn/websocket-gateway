Gateway cursor protocol (from SDK and multimode test client):
```
// Subscribe (call on mount / channel change, when connectionState === 'connected')
sendMessage({ service: 'cursor', action: 'subscribe', channel, mode: 'freeform' })

// Unsubscribe (call on channel change / unmount)
sendMessage({ service: 'cursor', action: 'unsubscribe', channel })

// Freeform position update (throttle to ~50ms, do NOT send on every pixel)
sendMessage({ service: 'cursor', action: 'update', channel, position: { x, y }, metadata: { mode: 'freeform' } })

// Inbound: cursor:subscribed — initial cursor list
// message: { type: 'cursor', action: 'subscribed', channel, cursors: RemoteCursor[], mode: string }

// Inbound: cursor:update — remote user moved
// message: { type: 'cursor', action: 'update', cursor: { clientId, position: { x, y }, metadata: { mode, userInitials, userColor, ... } } }

// Inbound: cursor:remove — remote user disconnected
// message: { type: 'cursor', action: 'remove', clientId }
```

Color/initials helpers (same logic as 07-01 — duplicate here, shared util comes later):
```typescript
const COLOR_PALETTE = [
  '#FF6B6B', '#4ECDC4', '#45B7D1', '#96CEB4', '#FECA57',
  '#FF9FF3', '#54A0FF', '#5F27CD', '#00D2D3', '#FF9F43',
  '#1DD1A1', '#F368E0', '#3742FA', '#2F3542', '#FF3838'
];

export function clientIdToColor(clientId: string): string {
  let hash = 0;
  for (let i = 0; i < clientId.length; i++) {
    hash = clientId.charCodeAt(i) + ((hash << 5) - hash);
  }
  return COLOR_PALETTE[Math.abs(hash) % COLOR_PALETTE.length];
}

export function clientIdToInitials(clientId: string): string {
  return clientId.substring(0, 2).toUpperCase();
}
```

    - returns: { cursors: Map<string, RemoteCursor>, sendFreeformUpdate: (x: number, y: number) => void }

    }
