# Plan 14-01: Auth & Presence Gap Closure

## Objective

Close two gaps identified by v1.3 milestone audit:
1. AUTH-09 — useWebSocket ignores `cognitoToken` changes after mount; refreshed JWT never reaches the gateway
2. PRES-03 — `setTyping` from `usePresence` was discarded in App.tsx; local user's typing state was never broadcast

## Tasks

### Task 1: AUTH-09 — Token refresh reconnect (App.tsx)

**File:** `frontend/src/app/App.tsx`

Add a `useEffect` in `GatewayDemo` that watches `config.cognitoToken`. When the token changes and the connection is active, call `reconnect()` so the gateway receives the updated JWT.

```typescript
const prevTokenRef = useRef(config.cognitoToken);
useEffect(() => {
  if (prevTokenRef.current !== config.cognitoToken && connectionState === 'connected') {
    prevTokenRef.current = config.cognitoToken;
    reconnect();
  } else {
    prevTokenRef.current = config.cognitoToken;
  }
}, [config.cognitoToken, connectionState, reconnect]);
```

**Why this works:** `reconnect()` is a `useCallback` with `[connect]` dep; `connect` is a `useCallback` with `[config]` dep. When `auth.idToken` refreshes → `authenticatedConfig` changes → new `config` prop → new `connect` → new `reconnect`. The effect fires, detects the change, and re-opens the WebSocket with the fresh token.

### Task 2: PRES-03 — Wire setTyping to ChatPanel (App.tsx)

**File:** `frontend/src/app/App.tsx`

Destructure `setTyping` from `usePresence` and pass it as `onTyping` prop to `ChatPanel`:

```typescript
const { users: presenceUsers, setTyping } = usePresence({ ... });
// ...
<ChatPanel onTyping={setTyping} ... />
```

### Task 3: PRES-03 — onTyping handler in ChatPanel (ChatPanel.tsx)

**File:** `frontend/src/components/ChatPanel.tsx`

- Add `onTyping?: (isTyping: boolean) => void` to `ChatPanelProps`
- Add `typingTimerRef` for 2s debounce
- `handleKeyDown`: call `onTyping(true)`, reset 2s timer to `onTyping(false)` on idle
- `handleSend`: clear timer, call `onTyping(false)` before sending
- `useEffect` cleanup: clear timer on unmount
