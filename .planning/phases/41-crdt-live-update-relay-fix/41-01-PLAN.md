<objective>
Fix the CRDT live update relay protocol mismatch (MISS-B) so real-time collaborative edits reach all connected clients, and add the missing EVENT_BUS_NAME env var to social-api in docker-compose (MISS-A).

Purpose: Currently, `broadcastBatch()` sends `{type:'crdt', action:'operations', operations:[...]}` but `useCRDT.ts` checks `msg.type === 'crdt:update'` and expects `msg.update` as a base64 string. This means live collaborative edits are silently discarded -- users only see each other's changes after a reconnect snapshot load. Fixing this enables true real-time collaboration.

Output: Updated `crdt-service.js` with corrected broadcast format; updated `docker-compose.localstack.yml` with social-api EVENT_BUS_NAME.
</objective>
